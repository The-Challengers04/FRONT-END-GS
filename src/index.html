O que é o projeto?

O "AgroTech Tower" é um sistema de agricultura vertical inteligente que maximiza o uso do espaço urbano, permitindo o cultivo de alimentos de forma vertical em estruturas projetadas especificamente para esse fim.

Sobre

O projeto tem como base a ODS 2 da agenda da Onu de 2030. No qual, o nosso objetivo é combater a fome mundial, trazendo soluções tecnológicas inovadoras para a agricultura

Como funciona a SmartGrow?

  • Construir um sistema que consiste em estruturas verticais Inteligentes que são torres modulares com múltiplos níveis de cultivo vertical

  • Utilizar tecnologia inteligente, como sensores para monitorar condições ambientais e sistemas de iluminação e controle de temperatura

  • Utilizar sensores de umidade do solo, temperatura e umidade, luminosidade, pH e nível de água

  • Fazer com que os sensores permitem monitorar e controlar fatores-chave para otimizar o crescimento das plantas, como irrigação, nutrição e condições ambientais
